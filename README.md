# Weather App
A simple web-based Weather App build using HTML, CSS, and JavaScript.
It fetches and displays weather information for any city using public weather API.

## Description
This project allows users to check the current weather of a city.
The app displays temperature, weather conditions, and icons for a better visual experience.
It focuses on UI design and basic API integration using JavaScript.

## Features 
- Search for any city to get current weather information
- Displays temperature, weather condition, and weather icons
- Clean and simple UI using HTML and CSS
- Interactive functionality  with JavaScript

## Tech Stack 
- HTML5
- CSS3
- JavaScript 
- Weather API
- Google fontw for styling
